# Birthday Wish Maker 🎂🎁

Create personalized birthday wishes with gift animation, music, and memories.

## Features
- 🎁 Animated gift box opening
- 🎵 YouTube song autoplay
- 📸 Memory photo gallery
- 💌 Special message
- 🎨 4 theme colors
- 📱 Mobile responsive
- 🔗 Shareable link
- 💯 100% Free, No Backend

## Deploy to GitHub Pages

1. Create new repo: `birthday-wish-maker`
2. Upload all files
3. Settings → Pages → Deploy from main branch
4. Your site: `https://username.github.io/birthday-wish-maker/`

## How to Use

1. Open `index.html`
2. Fill: Name, Age, YouTube song, Message, Pictures
3. Click Generate
4. Copy link and share

## Flow
Gift 🎁 → Tap → Song plays → Happy Birthday → Memories → Message → Happy Birthday {Name}

## Tech Stack
- Pure HTML/CSS/JS
- YouTube IFrame API
- Base64 URL encoding
- No database needed
